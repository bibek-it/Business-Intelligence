1 Version 1.0: The Original Dashboard

v1.0 Initial Release

The first iteration of the Interviews And Submissions Dashboard was designed for ITsura Inc. with a focus on employee contributions and basic metrics visualization.

Employee Contribution Section:

Left sidebar showing individual employee performance metrics including interviews, submissions, and hires for each team member.

Basic Distribution Charts:

Four separate visualization modules showing Mode (Video/In-person), Level (Initial/Middle/Senior), Tier (1/2/3), and Status (Scheduled/Completed).

Top Metrics Display:

Three key metrics prominently displayed: Total Interviews (844), Total Submissions (2307), and Hired (17).

Date Display:

Full date display showing "Thursday, October 16, 2025" at the top of the dashboard.

2 Version 2.0: The Enhanced Dashboard

v2.0 Current Release

The redesigned dashboard for ITsura Inc. features a more modern interface with enhanced data visualization, new metrics, and improved user experience.

Conversion Funnel:

New visualization showing the recruitment conversion funnel with percentages (39.1%, 23.9%, 68.2%) for each stage of the hiring process.

Walk-in Interviews Metric:

New metric added to track walk-in interviews (231), providing insights into unscheduled candidate traffic.

Detailed Applicant Table:

Comprehensive table showing individual applicant details including Name, Company, Position, and Status for granular data analysis.

Redesigned Layout:

More intuitive and modern interface with improved data organization and visual hierarchy.

3 Key Improvements from v1.0 to v2.0

The evolution from version 1.0 to 2.0 brought significant enhancements to both functionality and user experience.

Visual Design & User Interface

Complete redesign with a more modern, clean interface. Improved color scheme, better spacing, and enhanced visual hierarchy make data easier to interpret at a glance.

Data Visualization

Added the conversion funnel visualization to provide insights into the recruitment pipeline. Improved charts for submission status and interview level distribution with more interactive elements.

New Metrics & Insights

Introduced the Walk-in Interviews metric to track unscheduled candidate traffic. Added conversion rates at each stage of the hiring process to identify bottlenecks.

Detailed Data Access

Added a comprehensive applicant details table that allows users to drill down into individual candidate information, replacing the employee contribution section with more actionable data.
